🎉 CHAT FEATURE SETUP - FINAL STEP
=====================================

To enable the chat feature, you need to register the message routes in your backend server.

📝 STEP-BY-STEP INSTRUCTIONS:
------------------------------

1. Open your backend server file:
   - Usually: backend/server.js OR backend/index.js OR backend/app.js

2. Add these two lines:

   // Add this with other imports at the top
   const messageRoutes = require('./routes/messages');

   // Add this with other routes (after auth routes, donation routes, etc.)
   app.use('/api/messages', messageRoutes);

3. Save the file and restart your backend server

4. Test the chat feature:
   - Login as a donor
   - Create a donation
   - Login as a volunteer (different browser/incognito)
   - Accept the donation
   - Click "Chat with Donor" button
   - Send a message!
